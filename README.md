# Project Title: TODO API
 
## Frameworks used:
* Node.js
* Express
* MongoDB

## The API will handle the following:
* List all tasks created
* Get a specific task
* Delete a specified task
* Edit the title or description of a specific task

## List of Endpoints
* POST /v1/tasks
* GET /v1/tasks
* GET /v1/tasks/{id}
* DELETE /v1/tasks/{id}
* PUT /v1/tasks/{id}
